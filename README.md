Inuitcss Generator
===

Add a new line to main.scss and crete the related file at right place with template.

## Installation

  `npm install inuitcss-generator`

## Usage

    var inuitcss = require('inuitcss-generator');
    var file = inuitcss('main.scss', 'components', 'carousel');

## Tests

  `npm test`
